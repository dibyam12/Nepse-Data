API Documentation

Free REST API for Nepal Stock Exchange (NEPSE) data

🚀 Getting Started

The NEPSE Data API is completely free and requires no authentication. Simply make HTTP GET requests to any endpoint below.

Base URL: {{ request.scheme }}://{{ request.get_host }}/api/

📡 Endpoints

GET /api/

API overview with links to all endpoints.

GET /api/stocks/

List all available stock symbols with their latest close price and record count.

{
  "count": 250,
  "results": [
    {
      "symbol": "NABIL",
      "category": "stock",
      "latest_date": "2026-02-25",
      "latest_close": 1250.00,
      "total_records": 1500
    },
    ...
  ]
}
GET /api/stocks/{SYMBOL}/

Get historical OHLCV data for a specific stock (paginated, 100 per page).

Query Parameters:

  • from — Start date (YYYY-MM-DD)
  • to — End date (YYYY-MM-DD)
  • page — Page number
GET /api/stocks/NABIL/?from=2025-01-01&to=2025-12-31

GET /api/stocks/{SYMBOL}/latest/

Get the most recent price data for a single stock.

GET /api/latest/

Get the latest trading day's prices for all stocks at once.

GET /api/index/

Get NEPSE index history (defaults to last 30 entries).

Query Parameters:

  • from — Start date
  • to — End date
  • limit — Number of records (default: 30)

⚡ Rate Limits

Tier Limit
Anonymous 200 requests / hour

💻 Usage Examples

Python

import requests

# Get NABIL stock data
response = requests.get("{{ request.scheme }}://{{ request.get_host }}/api/stocks/NABIL/")
data = response.json()

for record in data["results"]:
    print(f"{record['date']}: Close={record['close']}, Volume={record['volume']}")

JavaScript

// Fetch latest prices
const response = await fetch("{{ request.scheme }}://{{ request.get_host }}/api/latest/");
const data = await response.json();

data.results.forEach(stock => {
  console.log(`${stock.symbol}: NPR ${stock.close} (${stock.change_percent}%)`);
});

cURL

curl "{{ request.scheme }}://{{ request.get_host }}/api/stocks/NABIL/?from=2025-01-01&to=2025-06-30"
